# Инструкция по Git

## Общие положения:

*Git* — это набор консольных утилит, которые отслеживают и фиксируют изменения в файлах.


*Git нужен для:*
* отслеживания изменений, произошедших с проектом.
* одновременной работы нескольких специалистов над одним проектом.

## Базовые команды

* git init
* git status
* git commit
